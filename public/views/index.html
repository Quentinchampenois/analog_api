Welcome on the Analog API

This API is written with Go and fully open source on the Github official repository

Available routes :

Index
  • GET - / => Index
Cameras
  • GET - /camera => Get all cameras
  • POST - /camera => Create new camera
  • GET - /camera/:id => Fetch a specific camera
  • PUT - /camera/:id => Update a camera
  • DELETE - /camera/:id => Delete a camera
Types
  • GET - /type => Get all types
  • POST - /type => Create new type
  • GET - /type/:id => Fetch a specific type
  • PUT - /type/:id => Update a type
  • DELETE - /type/:id => Delete a type
Films
  • GET - /film => Get all films
  • POST - /film => Create new film
  • GET - /film/:id => Fetch a specific film
  • PUT - /film/:id => Update a film
  • DELETE - /film/:id => Delete a film
Authentication
  • POST - /signup => Create a new account
  • POST - /signin => Sign in
User cameras
  • GET - /user/cameras => Get all cameras for current user
  • POST - /user/cameras/ => Register a new camera for current user
  • DELETE - /user/cameras/:id => Delete a camera for current user
Camera films
  • GET - /camera/films/:id => Get all films for the given camera
  • POST - /camera/films => Register a new film for the camera
  • PUT - /camera/films/:id => Rewind film for the camera
